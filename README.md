# Gulp 4 готовая сборка.

use node version 19

folder **sass-lite** - without JavaScript

folder **sass** - with JavaScript

folder **sass-slick** - with slick-slider and with jquery

### Project setup

```
npm install
```
### For develop
```
gulp
```
### For production
```
gulp build
```
